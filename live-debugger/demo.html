How to use this demo

Click the Debug button in the header to open the console panel. Then interact with the elements below. Every click, keystroke, and network request is logged in real-time.

Click any button to see it logged with element details.

Submit this form to see form events captured.

Trigger fetch requests to see the full request/response cycle.
